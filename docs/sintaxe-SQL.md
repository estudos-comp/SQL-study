<h1 align="center">SINTAXE SQL</h1>

<i><p align="center"> A sintaxe SQL é o conjunto de regras que define como escrever comandos para gerenciar bancos de dados relacionais. É padrão escrever instruções em letras maiúsculas e nomes de tabelas/colunas em minúsculas. Cada instrução deve terminar com um ponto e vírgula</p></i>

---

<h2 align="center">Instruções SQL</h2>

Um banco de dados geralmente contém uma ou mais tabelas. Cada tabela é
identificada por um nome e contém registros (linhas) com dados.

<h3 align="center">Alguns comandos SQL mais importantes</h3>

- `SELECT` &rarr; Extrai dados de um banco de dados
- `UPDATE` &rarr; Atualiza dados em um banco de dados
- `DELETE` &rarr; Exclui dados de um banco de dados
- `INSERT INTO` &rarr; Insere novos dados em um banco de dados
- `CREATE DATABASE` &rarr; Cria um nonvo banco de dados
- `ALTER DATABASE` &rarr; Modifica um banco de dados
- `CREATE TABLE` &rarr; Cria uma nova tabela
- `ALTER TABLE` &rarr; Modifica uma tabela
- `DROP TABLE` &rarr; Exclui uma tabela
- `CREATE INDEX` &rarr; Cria um índice (chave de pesquisa)
- `DROP INDEX` &rarr; Exclui um índice

## Consulta básica

```sql
SELECT coluna1, coluna2
FROM tabela
WHERE condicao;
```

## Todas as colunas

```sql
SELECT *
FROM tabela;
```

## Valores distintos

```sql
SELECT DISTINCT coluna
FROM tabela;
```

## Alias

```sql
SELECT coluna AS novo_nome
FROM tabela;
```

## Filtros

```sql
WHERE coluna = valor
WHERE coluna <> valor
WHERE coluna > valor
WHERE coluna BETWEEN a AND b
WHERE coluna LIKE 'Maria%'
WHERE coluna LIKE '%Maria%'
WHERE coluna IS NULL
WHERE coluna IS NOT NULL
WHERE coluna IN ('A', 'B')
```

## Join explícito

```sql
SELECT a.coluna, b.coluna
FROM tabela_a a
INNER JOIN tabela_b b ON a.id = b.id_a;
```

## Left join

```sql
SELECT a.coluna, b.coluna
FROM tabela_a a
LEFT JOIN tabela_b b ON a.id = b.id_a;
```

## Autojunção

```sql
SELECT f1.nome, f1.estilo, f2.estilo
FROM filme f1
INNER JOIN filme f2
        ON f1.nome = f2.nome
       AND f1.estilo <> f2.estilo;
```

## Agregação

```sql
SELECT coluna_grupo, COUNT(*) AS total
FROM tabela
GROUP BY coluna_grupo;
```

```sql
SELECT coluna_grupo, SUM(valor) AS total
FROM tabela
WHERE condicao_linha
GROUP BY coluna_grupo
HAVING SUM(valor) > 100
ORDER BY coluna_grupo;
```

## Funções agregadas

```sql
COUNT(*)
COUNT(coluna)
COUNT(DISTINCT coluna)
SUM(coluna)
AVG(coluna)
MAX(coluna)
MIN(coluna)
```

## Subconsulta escalar

```sql
SELECT nome, salario
FROM funcionario
WHERE salario > (
    SELECT AVG(salario)
    FROM funcionario
);
```

## `IN` com subconsulta

```sql
SELECT nome
FROM departamento
WHERE numero IN (
    SELECT num_dep
    FROM projeto
    WHERE local = 'RJ'
);
```

## `EXISTS`

```sql
SELECT f.nome
FROM funcionario f
WHERE EXISTS (
    SELECT *
    FROM dependente d
    WHERE d.codf = f.codf
);
```

## `CASE`

```sql
SELECT nome,
       CASE
           WHEN salario > 4000 THEN 'Alto'
           WHEN salario BETWEEN 2000 AND 4000 THEN 'Médio'
           ELSE 'Baixo'
       END AS faixa
FROM funcionario;
```

## `CREATE TABLE`

```sql
CREATE TABLE tabela (
    id INTEGER PRIMARY KEY,
    nome VARCHAR(50) NOT NULL,
    cpf CHAR(11) UNIQUE,
    idade INTEGER CHECK (idade >= 0)
);
```

## Chave estrangeira

```sql
FOREIGN KEY (id_cliente)
REFERENCES cliente(id_cliente)
ON DELETE CASCADE
ON UPDATE RESTRICT
```

## `ALTER TABLE`

```sql
ALTER TABLE tabela ADD coluna INTEGER;
ALTER TABLE tabela DROP coluna;
ALTER TABLE tabela MODIFY coluna VARCHAR(40);
```

## `DROP TABLE`

```sql
DROP TABLE tabela;
DROP TABLE tabela RESTRICT;
DROP TABLE tabela CASCADE;
```

## Autoincremento

```sql
-- Padrão moderno
id INTEGER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY

-- MySQL
id INT AUTO_INCREMENT PRIMARY KEY

-- SQL Server
id INT IDENTITY(1,1) PRIMARY KEY

-- PostgreSQL antigo
id SERIAL PRIMARY KEY
```

INSERT INTO tabela (coluna1, coluna2)

## Erros que mais derrubam ponto

| Erro | Correção |
|---|---|
| `WHERE coluna = NULL` | `WHERE coluna IS NULL` |
| `UPDATE tabela SET campo = valor` sem intenção global | Adicionar `WHERE`. |
| `DELETE FROM tabela` sem intenção global | Adicionar `WHERE`. |
| Coluna não agregada fora do `GROUP BY` | Agregar ou colocar no `GROUP BY`. |
| Produto cartesiano acidental | Adicionar condição de junção. |
| Filtrar agregação no `WHERE` | Usar `HAVING`. |
| Confundir `DELETE` e `DROP` | `DELETE` remove linhas; `DROP` remove tabela. |
